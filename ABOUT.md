# About FrpNexus

FrpNexus 是一个本地优先的 FRP 桌面部署与运维控制台。

它面向个人开发者、自托管用户和轻量运维场景，目标是把远程 Linux 节点管理、FRP 核心部署、TOML 配置生成、进程控制、日志查看和基础诊断放到一个清晰的桌面工具里。

## 为什么做 FrpNexus

FRP 很强大，但一次完整的部署和排障通常会牵涉多件事：下载合适的 Release、编写配置、上传文件、登录服务器、启动进程、查看日志、定位错误。

FrpNexus 希望把这些分散步骤收拢到一个本地应用中，让用户围绕“让一个 FRP 服务或隧道真正跑起来”完成闭环。

## 产品边界

FrpNexus 不是：

- 云平台
- Web Dashboard
- SaaS 后台
- 团队协作系统
- 计费或商业化平台
- 单纯的 TOML 文本编辑器

它更像一个专业、克制、开发者友好的 Windows 桌面运维工具。

## 优先用户

FrpNexus 优先服务：

- 有一台或多台 VPS 的个人开发者
- 想远程访问家中 NAS、开发机或内网服务的用户
- 想临时暴露本地 Web/API 服务的人
- 自托管爱好者
- 熟悉基础网络概念，但希望降低重复配置和 SSH 操作成本的人

## 当前方向

项目当前聚焦 FRP 的桌面端部署与运维主流程：

- 节点管理
- SSH/SFTP 连接
- FRP 核心文件管理
- TOML 配置生成与上传
- 远程进程控制
- 日志查看与基础诊断
- 本地设置、安全认证和数据管理

项目保持本地优先、开源透明，不强制依赖云端账号。
